# My localization contributions

Here are some projects that contain localizations made by myself, especially on Chinese (Hong Kong) `zh-HK` (sort by date):

- [UTM](https://github.com/utmapp/UTM), virtual machines for iOS and macOS
- [CrystalFetch](https://github.com/TuringSoftware/CrystalFetch), macOS UI for creating Windows installer ISO from UUPDump
- [Equinox](https://github.com/rlxone/Equinox), tool for creating dynamic wallpapers for macOS
- [AltTab](https://github.com/lwouis/alt-tab-macos), Windows alt-tab on macOS
- [CotEditor](https://github.com/coteditor/CotEditor), lightweight Plain-Text Editor for macOS
